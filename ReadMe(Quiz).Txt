        The code asks the user for information about them including their name and ID number. Their ID number input is checked for validity. The user has an option to take a 10 or 20 question quiz. Depending on what they pick, their quiz with the chosen number of questions is generated for them. They’re asked questions with possible answers one-at-a-time and answer them. Their answers are checked for validity and if they are correct, their score increases. At the end of the test a report is printed with their information and all of the questions with the correct answers. The user has the option to clear the screen and start a new test to quit.
